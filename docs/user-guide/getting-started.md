# Getting Started

## Installation

Extract the theWheel distribution and run `theWheel.exe`. No installation required.

## Opening a Space

1. Go to **File → Open**
2. Select a `.spx` file (sample files are included in the `data/` directory)
3. The space will load and display its nodes

## Navigating

- **Click** a node to activate it — related nodes grow larger as activation spreads
- **Double-click** a node to expand its children
- **Drag** a node to reposition it
- **F1** — open this help

## Creating a New Space

1. Go to **File → New**
2. Right-click in the tree view to add nodes
3. Use **File → Save** to save as `.spx`
